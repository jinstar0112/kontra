Kontra.keys

A keyboard API. You can use it move the main sprite or respond to a key press.

The keyboard API can handle key combinations, multiple keys that must be pressed together, by separating each key with a +. For example, the copy function ctrl+c can be a key combination written the same way.

function update() {
  if (kontra.keys.pressed('left')) {
    // move left
  }

  if (kontra.keys.pressed('ctrl+c')) {
    // copy
  }
}

Available keys#

Below is a list of keys that you can use. If the key requires the shift key (such as !), the shift key will automatically be added as a required key combination (shift+1).

This means that any shift key is equivalent to the key combination of shift + the non-shift key (! is equivalent to shift+1).

  • a-z
  • 0-9
  • f1-f20
  • numpad0-numpad9
  • backspace, tab, enter, shift, ctrl, alt, capslock, esc, space, pageup, pagedown, end, home ,left, up, right, down, insert, delete, leftwindow, rightwindow, select, numlock, scrolllock
  • *+-./;=,\`[]'
  • ~!@#$%^&*(){}:"<>?

kontra.keys.bind(key)#

key {string | string[]}
Keys or key combinations to bind.
callback {function}
Function to call when the key or key combination is pressed.

Bind a set of keys or key combinations to a function when pressed. Takes a single key or key combination or an array of keys or key combinations.

kontra.keys.bind('p', function() {
  // pause the game
});
kontra.keys.bind('ctrl+c', function() {
  // copy
});
kontra.keys.bind(['enter', 'space'], function() {
  // fire gun
});

kontra.keys.pressed(key)#

key {string}
Key or key combination.

Returns true if the key or key combination is pressed. Use during an update() function to perform actions each frame.

kontra.keys.pressed('a');  //=> returns true if 'a' is pressed
kontra.keys.pressed('ctrl+c');  //=> returns true if the 'ctrl' and 'c' keys are pressed

kontra.sprite({
  update: function() {
    if (kontra.keys.pressed('left')){
      // left arrow pressed
    }
    else if (kontra.keys.pressed('right')) {
      // right arrow pressed
    }

    if (kontra.keys.pressed('up')) {
      // up arrow pressed
    }
    else if (kontra.keys.pressed('down')) {
      // down arrow pressed
    }
  }
});

kontra.keys.unbind(key)#

key {string | string[]}
Keys or key combinations to unbind.

Unbind a bound set of keys or key combinations. The bound function will no longer be called when the keys are pressed. Takes a single key or key combination or an array of keys or key combinations.

kontra.keys.unbind('left');
kontra.keys.unbind('ctrl+c');
kontra.keys.unbind(['enter', 'space']);
