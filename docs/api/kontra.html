kontra.canvas#

{HTMLCanvasElement}

The main canvas element of the game. Use this property to easily get the width and height of the canvas.

kontra.context#

{CanvasRenderingContext2D}

The main 2D context of the canvas element. Use this property to easily get the main context for the game.

kontra.init([canvas])#

canvas {string | HTMLCanvasElement}
Optional. The ID of the canvas or a Canvas element

Initialize the game and set up internal references to the canvas and drawing context. By default, it will use the first canvas element on the page.

You can also pass the ID of the canvas or a Canvas element to use as the main drawing context.
