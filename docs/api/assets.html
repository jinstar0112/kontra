Kontra.assets

A promise based asset loader.

kontra.assets.load('/imgs/character.png', '/data/tile_engine_basic.json', ['/audios/music.ogg', '/audios/music.mp3'])
  .then(function() {
    // all assets have loaded
  });

kontra.assets.audioPath#

Base path for all audio assets. If a base path is set, all load calls for audio assets will prepend the base path to the URL.

kontra.assets.audioPath = '/audio';
kontra.assets.load('music.ogg');  // no need to add the /audio directory

kontra.assets.audios#

Object of all loaded audios by both file name and path. If audioPath was set before the audio was loaded, the path will not include the audio base path.

kontra.assets.load('/audios/music.ogg')
  .then(function() {
    // Audio file can be accessed at both
    // kontra.assets.audios['/audios/music'] and
    // kontra.assets.audios['/audios/music.ogg']
  });

kontra.assets.audioPath = '/audio';
kontra.assets.load('sound.ogg')
  .then(function() {
    // Audio file can be accessed at both
    // kontra.assets.audios.sound and
    // kontra.assets.audios['sound.ogg']
  });

kontra.assets.data#

Object of all loaded data files by both file name and path. If dataPath was set before the data files was loaded, the path will not include the data base path.

If the data file is a json file, it will automatically parse the file and save it as an object.

kontra.assets.load('/data/file.txt')
  .then(function() {
    // Data file can be accessed at both
    // kontra.assets.data['/data/file'] and
    // kontra.assets.data['/data/file.txt']
  });

kontra.assets.dataPath = '/data';
kontra.assets.load('info.json')
  .then(function() {
    // Data file can be accessed at both
    // kontra.assets.data.info and
    // kontra.assets.data['info.json']
  });

kontra.assets.load(url[, ...])#

url {string | string[]}
The path or paths to the file.

Load Image, Audio, or data files using promises. Calls loadImage(), loadAudio, and loadData() based on each asset type. You can pass it as many assets as you need.

When loading audio assets, you can specify multiple formats and the loader will determine which format to load based on the current browser support. Order of the array designates audio preference.

Once a file is loaded, it will be accessible by it's file name and path from either the kontra.assets.audios, kontra.assets.data, or kontra.assets.images objects, depending on it's type.

kontra.assets.load('/imgs/character.png', '/data/tile_engine_basic.json', ['/audio/music.mp3', '/audio/music.ogg'])
  .then(function() {
    /* ... */
  });

kontra.assets.loadAudio(url)#

url {string}
The path to the file.

Load an Audio asset using promises. Once loaded, the asset is available by it's file name and the passed in URL under kontra.assets.audios.

kontra.assets.loadData(url)#

url {string}
The path to the file.

Load a data asset using promises. Once loaded, the asset is available by it's file name and the passed in URL under kontra.assets.data.

kontra.assets.loadImage(url)#

url {string}
The path to the file.

Load an Image asset using promises. Once loaded, the asset is available by it's file name and the passed in URL under kontra.assets.images.
