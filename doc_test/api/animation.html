Animation(​properties)

An object for drawing sprite sheet animations.

An animation defines the sequence of frames to use from a sprite sheet. It also defines at what speed the animation should run using frameRate.

Typically you don't create an Animation directly, but instead would create it through a SpriteSheet.

import { SpriteSheet, Animation } from 'kontra';

let spriteSheet = SpriteSheet({
  // ...
});

let animation = Animation({
  spriteSheet: spriteSheet,
  frames: [1,2,3,6],
  frameRate: 30
});

Animation Parameters

properties

Object. Properties of the animation.

properties.spriteSheet

SpriteSheet. Sprite sheet for the animation.

properties.frames

An Array of Numbers. List of frames of the animation.

properties.frameRate

Number. Number of frames to display in one second.

properties.loop Optional

Boolean. If the animation should loop. Defaults to true.

Animation​.frameRate #

Number. Number of frames to display per second. Adjusting this value will change the speed of the animation.

Animation​.frames #

An Array of Numbers. Sequence of frames to use from the sprite sheet.

Animation​.height #

Number. The height of an individual frame. Taken from the property of the same name in the spriteSheet.

Animation​.loop #

Boolean. If the animation should loop back to the beginning once completed.

Animation​.margin #

Number. The space between each frame. Taken from the property of the same name in the spriteSheet.

Animation​.render(properties) #

Draw the current frame. Used when the animation is not stopped.

render Parameters

properties

Object. How to draw the animation.

properties.x

Number. X position to draw.

properties.y

Number. Y position to draw.

properties.width Optional

Number. width of the sprite. Defaults to Animation.width.

properties.height Optional

Number. height of the sprite. Defaults to Animation.height.

properties.context Optional

Canvas​Rendering​Context2D. The context the animation should draw to. Defaults to core.getContext().

Animation​.reset() #

Reset an animation to the first frame.

Animation​.spriteSheet #

SpriteSheet. The sprite sheet to use for the animation.

Animation​.update([dt]) #

Update the animation. Used when the animation is not paused or stopped.

update Parameters

dt Optional

Number. Time since last update. Defaults to 1/60.

Animation​.width #

Number. The width of an individual frame. Taken from the property of the same name in the spriteSheet.
